docker run -d 是 Docker 命令的一部分，用于在后台（detached mode）运行容器。以下是该命令的基本语法和一些常见选项：

bash Copy code docker run -d [OPTIONS] IMAGE [COMMAND] [ARG...] -d：以后台（守护进程）模式运行容器。 OPTIONS：其他选项，例如端口映射、数据卷挂载等。 IMAGE：指定要运行的容器镜像。 COMMAND：在容器内运行的命令。 ARG：命令的参数。 以下是一些示例：

运行一个基本的容器：

bash Copy code docker run -d ubuntu 这将在后台运行一个基本的 Ubuntu 容器。

运行容器并映射端口：

bash Copy code docker run -d -p 8080:80 nginx 这将在后台运行一个 Nginx 容器，并将宿主机的8080端口映射到容器的80端口。
